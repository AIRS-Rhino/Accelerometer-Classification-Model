# Accelerometer-Classification-Model
This repository is associated with the article titled "Accelerometry as a method of behavior observation in ex situ African rhinoceros". Manuscript was submitted to PLOS One. Code was written by Allie Richardson. Co-authors include Andrew Alba, Gina Ferrie, Emily Bach, Christopher Robinson, Jessica Rivera, Lara Metrione, and Mandi Schook. 

The code requires an older version of the 'xgboost' package, as newer versions are no longer compatible with the 'rabc' package. To ensure the code runs correctly, install 'xgboost 1.7.11.1' instead of the default version. This change in versions was not necessary when the code was originally written, but became necessary after various updates to the involved packages.
